## Details

_boolean_ **Content.StartScan()** _()_

Start a content scan.

Script Permissions: content

Return Values:

| Value | Description                                                |
| ----- | ---------------------------------------------------------- |
| true  | Successfully started a scan.                               |
| false | Failed to start a scan. A scan may already be in progress. |
