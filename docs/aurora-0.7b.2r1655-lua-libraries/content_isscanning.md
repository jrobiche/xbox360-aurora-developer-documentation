## Details

_boolean_ **Content.IsScanning()** _()_

Check if a content scan is in progress.

Script Permissions: content

Return Values:

| Value | Description               |
| ----- | ------------------------- |
| true  | A scan is in progress.    |
| false | There is no scan running. |
