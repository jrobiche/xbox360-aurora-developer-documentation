## Details

_unsigned_ **Dvd.GetTrayState** _()_

Get the tray state of the DVD tray.

Script Permissions: filesystem

Return Values:

| Value | Description |
| ----- | ----------- |
| 1     | Closing     |
| 2     | Open        |
| 3     | Opening     |
| 4     | Closed      |
| 5     | Error       |
